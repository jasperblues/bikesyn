fileNameFor

  • fileNameFor(dirNameOrPath: string, extension?: undefined | string, resourceName?: undefined | string): string
    • Defined in liberation-data/drivine/src/DrivineInjectionDecorators.ts:49

    Resolves a resource name given the arguments. The single argument version is for use when a TypeScript path alias is set up. Example @Inject('@/traffic/routesBetween'). This is the original behavior in v1.x, however some users found it puzzling.

    The two argument version is for when there is not path alias, for example: @Inject(__dirname, 'routesBetween').

    Parameters

    • dirNameOrPath: string

      Either a directory name or the full path.

    • Optional extension: undefined | string

      The extension of the file, if any.

    • Optional resourceName: undefined | string

    Returns string
